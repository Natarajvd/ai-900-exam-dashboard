Question Format Guide

Create exam files with .md or .txt format

Quick Start

Every exam file needs:

  • # Exam Title — file title (one # hash)
  • ## Module Name — domain/topic (two ## hashes)
  • ### Q1, Q2, Q3... — questions (three ### hashes)
  • 4 options (A, B, C, D) — exactly 4 choices
  • Correct answer: B — which option is right

Markdown Example 

# Azure Fundamentals Exam

## Cloud Concepts

### Q1
What is cloud computing?

- **A.** Using physical servers
- **B.** Internet-based services
- **C.** Local storage only
- **D.** Install software locally

**Correct answer:** B

**Explanation:** Cloud computing delivers computing services over the internet.

Text (.txt) Format 

# Python Quiz

## Data Types

### Q1
Which is immutable?

- A. List
- B. Dictionary
- C. Tuple
- D. Set

**Answer:** C

**Explanation:** Tuples cannot be modified after creation.

Accepted Formats

Options
  • - **A.** text
  • - A. text
  • A) text
Answer
  • **Correct answer:** B
  • **Answer:** B
  • Answer: B

Important Rules

✅ Do This
  • ✓ Use .md or .txt
  • ✓ Exactly 4 options
  • ✓ Plain text only
  • ✓ Add explanations
❌ Avoid This
  • ✗ HTML tags
  • ✗ Fewer than 4 options
  • ✗ .docx or .pdf files
  • ✗ Missing answers
